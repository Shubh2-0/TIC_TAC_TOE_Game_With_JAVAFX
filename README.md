## 🎯 About

Classic **Tic Tac Toe** game developed using JavaFX. Enjoy the timeless game with a modern graphical interface!

## ✨ Features

- 🎲 **Two Player Mode** - Play with a friend
- 🎨 **Interactive UI** - Click to place X or O
- 🔄 **Auto Restart** - New game after win/draw
- ⚠️ **Error Handling** - Can't select occupied cells
- 🏆 **Win Detection** - Automatic winner announcement

## 🎮 Game Rules

```
 X │ O │ X
───┼───┼───
 O │ X │ O     → Get 3 in a row to win!
───┼───┼───
 X │ O │ X
```
